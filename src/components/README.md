# Components

Tento priečinok obsahuje znovupoužiteľné React komponenty.

## Štruktúra

Každý komponent by mal mať svoj vlastný priečinok s:
- `ComponentName.jsx` - hlavný súbor komponentu
- `ComponentName.module.css` - štýly (ak používate CSS modules)
- `index.js` - export komponentu

## Príklad

```
components/
  Button/
    Button.jsx
    Button.module.css
    index.js
```
